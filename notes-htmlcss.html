Fonts

Font stack, like font-family: Constantia, Georgia, Serif

It ends with a generic font family (serif). This is great because it ensures that even if the browser can't find the other two specific fonts, it will still display the general kind of font we want.


Sequence of FONT in CSS:
font_style font_size font_family;
e.g. h1{ font: bold italic 14pt Impact, "Arial Black", sans-serif; }
